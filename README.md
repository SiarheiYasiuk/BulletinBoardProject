Проект "Доска объявлений".

Описание проекта:

Доска объявлений - это веб-приложение, которое предоставляет пользователю возможность просматривать и размещать объявления на различные темы. Пользователи могут просматривать доступные объявления, осуществлять поиск по категориям, фильтровать объявления по цене, сортировать по новизне и стоимости, доступен подробный просмотр каждого объявления и, авторизованным пользователям, предоставлена возможность редактировать собственные объявления, а так же добавлять новые. Так же пользователю доступны регистрация, авторизация и личный кабинет, где он может редактировать данные о своей учётной записи.


Основные функции и особенности проекта:

- Просмотр объявлений: Пользователи могут просматривать список доступных объявлений на главной странице. Каждое объявление содержит информацию о названии, цене, местоположении и дате размещения.
- Поиск по категориям: Пользователи могут искать объявления по определенным категориям или ключевым словам. Это помогает быстро найти интересующие объявления и сузить результаты поиска.
- Фильтрация по цене: Пользователи могут применять фильтры для ограничения результатов по цене. Например, они могут указать минимальную и максимальную цену, чтобы получить объявления, соответствующие их бюджету.
- Сортировка по цене и новизне: Пользователи могут применять сортировку для сортировки результатов по цене и новизне. Например, они могут указать отсортировать объявление по новизне чтобы сначала увидеть самые новые объявления.
- Пагинация: Для удобства навигации по длинным спискам объявлений, реализована пагинация, которая разбивает результаты на страницы.
- Авторизация и регистрация: Для создания своих собственных учётных записей и последующего сохранения данных в них, была добавлена возможность регистрации и последующей авторизации пользователей.
- Личный кабинет: Пользователи посредством навигации могут перемещаться в личный кабинет, где их будет встречать форма, позволяющая редактировать данные пользователя - такие, как город, имя, телефон.
- Подробный просмотр объявлений: Пользователи посредством нажатия на объявление в списке объявлений могут переходить на страницу с подробной информацией об объявлении.
- Добавление объявлений: При осуществлении взаимодействия с кнопкой "Подать объявление", авторизованные пользователю могут добавлять объявление к уже существующим.
- Изменение объявлений: Пользователи имеют возможность изменять свои собственные объявления путём нажатия на подробную информацию про своё объявление и, далее нажать на кнопку управления объявлением, после чего выбрать изменение.


Инструкция по установке и запуску проекта:

1. Установить все зависимости командой "npm install";
2. В файле .env изменить по желанию переменную "NODE_ENV" = "значение", где в роли значения могут выступать:
    - "production" для запуска сборки в режиме production;
    - "development" для запуска сборки в режиме development;
3. Создать новый терминал в среде разработки и прописать там команду "npm start" для запуска.

При правильно выполненных и успешных действиях, в браузере откроется окно с главной страницей нашего проекта.


Технические детали:

Проект "Доска объявлений" реализуется с использованием веб-технологий, включая HTML, CSS и JavaScript. Данные объявлений хранятся на сервере в базе данных, и взаимодействие с сервером осуществляется посредством API-запросов.


Используемые технологии:
- HTML;
- CSS;
- JavaScript;
- React;
- npm.
